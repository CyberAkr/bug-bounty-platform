📝 Rédiger un bon rapport

Pour qu’un rapport soit accepté rapidement, il doit être :

  • 🧾 Clair et structuré – pas de texte brouillon
  • 🔍 Accompagné de preuves – screenshot, log, lien, vidéo…
  • 📌 Respectueux du périmètre défini par l’entreprise
  • 🕵️‍♂️ Reproductible facilement – étapes concrètes pour tester
